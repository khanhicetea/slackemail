# SlackEmail

Receive email as slack notification via webhook (Slack webhook compatible)

## TL;DR

Using Docker to run

```bash
$ docker run -d -p 3000:3000 khanhicetea/slackemail
```

Then open **http://localhost:3000** to try a shot !

## Install & Start

```bash
$ npm install
$ npm start
```

## Now.sh Deploying

```bash
$ now
```
